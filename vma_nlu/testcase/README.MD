# Test Cases Document

### Usage
- Test template : 
```
from vma_nlu.tester import Tester

tester = Tester()
tester.test_template()

# Test NER
tester.test_person_name()
tester.test_time()
tester.test_date()

# Test VLSP2016
tester.test_person_name_vlsp('train')
tester.test_person_name_vlsp('dev')
tester.test_person_name_vlsp('test')
```
